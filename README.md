# 🤖 Prime Studios - Telegram Bots Collection

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Telegram API](https://img.shields.io/badge/Telegram_API-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)

> Um mono-repositório contendo uma suíte de robôs para Telegram desenvolvidos pela **Prime Studios**. Focados em automação de processos, atendimento ao cliente, vendas e gestão comercial.

## 📦 Coleção de Bots

Cada pasta neste repositório contém um bot independente projetado para resolver um problema de negócios específico:

| Bot | Pasta | Descrição Principal |
| :--- | :--- | :--- |
| **Assistx** | `/Assistx` | Assistente virtual inteligente para atendimento ao cliente e suporte. |
| **Barber Pro** | `/Barber` | Sistema de agendamento e gestão para barbearias. |
| **Prime Draw** | `/PrimeDraw` | Automação para sorteios, engajamento e campanhas promocionais. |
| **Stock Control** | `/StockControl` | Gerenciamento de inventário, alertas de estoque e controle de entrada/saída via chat. |
| **Store Bot** | `/Store` | Vitrine virtual, catálogo de produtos e processamento de pedidos para e-commerce. |
| **Taxi Pro** | `/TaxiPro` | Solução para agendamento de corridas e despacho de motoristas. |
| **Vendas VIP** | `/VendasVip` | Funil de vendas exclusivo, checkout e relacionamento com clientes VIP. |

*(Nota: Alguns bots podem possuir variações ou versões atualizadas, como `Barbeiro_pro.py` e `BarberPro.py`, mantidas para fins de legado ou testes)*.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python 3.x
* **Bibliotecas Principais:** * Biblioteca de integração com Telegram (`telebot` / `aiogram` / `python-telegram-bot`)
  * `python-dotenv` para gerenciamento seguro de variáveis de ambiente.

## 🚀 Como Executar Localmente

### Pré-requisitos
* Python 3.8+ instalado.
* Um token de bot válido gerado através do [BotFather](https://t.me/botfather) no Telegram.

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/SEU_USUARIO/prime-telegram-bots.git](https://github.com/SEU_USUARIO/prime-telegram-bots.git)
   cd prime-telegram-bots
